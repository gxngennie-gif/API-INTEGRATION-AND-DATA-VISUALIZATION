#DISCRIPTION:
his project is developed using Visual Studio Code (VS Code).
The main objective of this project is to fetch real-time weather data from an online weather API and visualize it using graphs.
The project collects hourly weather data such as:
Temperature (°C)
Relative Humidity (%)
for Delhi using the Open-Meteo API.
The fetched data is processed and converted into readable time format. Then the data is plotted using graphs to analyze temperature and humidity variations throughout the day.

Tools & Technologies Used
1️. Programming Language
Python 3.13
2️. Code Editor
Visual Studio Code
3️. Python Libraries Used
requests → To fetch data from API
matplotlib → To create graphs
datetime → To format date and time
4️. API Used
Open-Meteo Weather API (for real-time weather data)

How The Project Works:
Sends a request to weather API
Receives weather data in JSON format
Extracts:
Time
Temperature
Humidity
Converts time into proper format
Plots two separate graphs:
Temperature vs Time
Humidity vs Time
